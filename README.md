# dashboard-vendas-xbox
Dashboard de vendas desenvolvido em Microsoft Excel durante o desafio da DIO.

## Sobre

Projeto desenvolvido durante o desafio da DIO utilizando Microsoft Excel.

O objetivo foi transformar uma base de dados de vendas em um dashboard visual e interativo utilizando Tabelas Dinâmicas, Gráficos Dinâmicos e Segmentações de Dados.

---

## Tecnologias

- Microsoft Excel
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- Segmentação de Dados (Slicers)

---

## Recursos

- Receita por Plano
- Receita por Tipo de Assinatura
- Receita por Mês
- Renovação Automática
- EA Play
- Minecraft Pass
